What is Angular?

  • full-featured SPA framework
  • open source ( GitHub, MIT )
  • foriginally written by Misko Hevery
  • extended html - declarative markup
  • speed!

SPAs

  • no page reloads
  • no redirections to other pages
  • performant ( slick interface and fast processing )
  • challenges:
    - Routing - History - Promises
    - View loading  - Object Modeling
    - Modules - Data Binding - Cache/Sync 

Why AngularJS?

  • separation of concerns 
    controllers, views, models(data) and services
  • two-way data binding and scopes
  • directives and filters ( simplified development )
  • modules ( code reuse )
  • dependency injection
  • tests ( Jasmine )
  • vibrant community

Directives

  • ‘A’ – Attribute (You want to use your directive as <div rating>)
  • ‘E’ – Element (Use it as <rating>)
  • ‘C’ – Class. (Use it like <div class=”rating”>)
  • ‘M’ – Comment (Use it like <!– directive: rating –>
